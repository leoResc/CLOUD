User

A user is a person who has logged in to your CLOUD. Each user has a free selectable nickname. As long as there exists no event for the current point in time, no user can log in. The password for each user is this one which the admin assigned to the current event.
The admin himself is a special user who can - of course - log in at any time. On this site, you can change the admin password and you can also delete user.
